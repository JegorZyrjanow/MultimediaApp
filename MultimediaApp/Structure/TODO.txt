	Элементы управления WPF
	https://www.tutorialspoint.com/wpf/wpf_controls.htm
	https://metanit.com/sharp/wpf/5.1.php

1. Список мемов
	|   ListBox, TreeView или другой подобный компонент со списком названий мемов.
	|   При выделении одного из элементов списка происходит отображение соответствующей 
	|   картинки в ImageBox:
	|
	✓  https://www.wpf-tutorial.com/basic-controls/the-image-control/
	
2. Хранение
	|	Хранение данных о мемах должно быть реализовано с помощью json или xml файла.
	|
	|	При этом в главном окне программы должны быть 2 кнопки:
	|	«Сохранить» и «Загрузить».
	|	При сохранении информация о добавленных мемах должна записаться в файл,
	|	при загрузке должен отобразиться список ранее сохраненных мемов.
	✓	Работа с JSON:

	https://metanit.com/sharp/tutorial/6.5.php
	https://docs.microsoft.com/ru-ru/dotnet/standard/serialization/system-text-jsonhow-to

3. Добавление мема в каталог
	|	Добавление мема в каталог через диалоговое окно:
	|	название мема, расположение на диске, категория и др. по желанию.
	✓	Расположение на диске нужно указывать с помощью OpenFileDialog:
	
	https://docs.microsoft.com/ruru/dotnet/api/system.windows.forms.openfiledialog?view=netcore-3.1
	https://metanit.com/sharp/windowsforms/4.20.php

4. Поиск по названию
	|	В главном окне поле для поиска по названию. 
	✓	в списке мемов должны оставаться только те мемы, которые содержат в названии введенный текст (без учета регистра).

5. Фильтрация по категории
	В главном окне ComboBox со списком категорий. 
	При выборе определенной в списке мемов должны оставаться мемы выбранной категории.

6. Удаление мема
	В главном окне должна присутствовать кнопка «Удалить мем». При нажатии 
	на эту кнопку, выделенный мем должен удаляться.
